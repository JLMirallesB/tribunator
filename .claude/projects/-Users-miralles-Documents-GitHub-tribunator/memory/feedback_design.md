Modern minimalist design based on whites, blacks, and greys. Color ONLY when it conveys information or highlights something.

Floor plan editor is NOT responsive (warn user on mobile). Tribunal/room queries ARE responsive.

**Why:** The user wants a professional, clean tool - not a colorful dashboard. Color should be a signal, not decoration.

**How to apply:** Use grey-scale for structure/chrome. Reserve blue for actions, green for available/free, red/amber for assigned/warnings. No emojis unless requested.
